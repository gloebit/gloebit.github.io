Money Module Details

We'll be testing the money module in beta with sandboxed payments until we feel the product is secure enough for live payments and cashouts, and the feature set is not missing any vital functionality.

Get all the details on the module.

We've done our best to build a full currency solution for OpenSim which is easy to install, configure, and use. Avatars will complete a simple, 1-step web authorization prior to their first transaction on a grid, and then all the built in OpenSim transactions should work with that user's Gloebit account and Gloebit balance.
