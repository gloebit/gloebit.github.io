Buy progress messages from Gloebit Money Module

The Gloebit Money Module reports the progress through the stages of the transaction. The messages include a shortened transaction id within square braces on every message. If a transaction fails, a failure message is delivered with the entire transaciton id.

The messages are also recorded in the chat history in case a user wants to go back to review them.

Buy record in transaction history on Gloebit

All transactions are recorded in your transaction history record.

If you expand a record by clicking on the description, you can see additional details.
